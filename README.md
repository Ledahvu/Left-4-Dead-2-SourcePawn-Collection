## 📖 English

### Overview
This repository contains a collection of **SourcePawn** source code (plugins/scripts) for **Left 4 Dead 2** running on **SourceMod**.  
These plugins enhance gameplay, fix common bugs, add admin tools, or customize server behavior.

### Requirements
- [SourceMod](https://www.sourcemod.net/) 1.11 or newer
- [Metamod:Source](https://www.sourcemm.net/) 1.11+
- Left 4 Dead 2 (latest stable build)

### Included Plugins (example)
| Plugin | Description |
|--------|-------------|
| `l4d2_cola_drop_heal.sp` | Use cola to heal |
| `l4d2_selfkill_ultimate.sp` | Use selfkill commands to commit suicide |
| `l4d2_bodytrap.sp` | Attach grenades to survivors or infected |

*(Replace with your actual plugins)*

### Installation
1. Compile `.sp` files using `spcomp` (included with SourceMod).
2. Upload compiled `.smx` files to `addons/sourcemod/plugins/`.
3. Restart the server or change map.

### Contributing
Feel free to open issues or pull requests. Please follow SourcePawn style conventions.

## 🇻🇳 Tiếng Việt

### Tổng quan
Kho lưu trữ này chứa mã nguồn **SourcePawn** (plugin/script) dành cho **Left 4 Dead 2** chạy trên **SourceMod**.  
Các plugin giúp cải thiện lối chơi, sửa lỗi phổ biến, thêm công cụ quản trị hoặc tùy chỉnh máy chủ.

### Yêu cầu
- [SourceMod](https://www.sourcemod.net/) 1.11 trở lên
- [Metamod:Source](https://www.sourcemm.net/) 1.11+
- Left 4 Dead 2 (bản mới nhất)

### Danh sách plugin (ví dụ)
| Plugin | Mô tả |
|--------|-------|
| `l4d2_cola_drop_heal.sp` | Sử dụng cola để hồi máu |
| `l4d2_selfkill_ultimate.sp` | Sử dụng các lệnh selfkil để tự sát |
| `l4d2_bodytrap.sp` | Gắn các loại lựu đạn lên người survivors hoặc infected |

*(Thay bằng plugin thực tế của bạn)*

### Cài đặt
1. Biên dịch file `.sp` bằng `spcomp` (có sẵn trong SourceMod).
2. Upload file `.smx` đã biên dịch vào thư mục `addons/sourcemod/plugins/`.
3. Khởi động lại máy chủ hoặc đổi map.

## 🇨🇳 中文

### 概述
本仓库包含用于 **Left 4 Dead 2** 的 **SourcePawn** 源代码（插件/脚本），运行在 **SourceMod** 环境下。  
这些插件可以改进游戏玩法、修复常见错误、添加管理工具或自定义服务器行为。

### 环境要求
- [SourceMod](https://www.sourcemod.net/) 1.11 或更高
- [Metamod:Source](https://www.sourcemm.net/) 1.11+
- Left 4 Dead 2（最新稳定版）

### 包含的插件（示例）
| 插件 | 说明 |
|------|------|
| `l4d2_cola_drop_heal.sp` | 用可乐疗伤 |
| `l4d2_selfkill_ultimate.sp` | 使用自杀指令自杀 |
| `l4d2_bodytrap.sp` | 将手榴弹绑在幸存者或感染者身上 |

*(请替换为您实际的插件列表)*

### 安装方法
1. 使用 `spcomp`（包含在 SourceMod 中）编译 `.sp` 文件。
2. 将编译好的 `.smx` 文件上传到 `addons/sourcemod/plugins/` 目录。
3. 重启服务器或更换地图。
